# neovim-config

An opinionated and minimalistic neovim configuration.

## Plugins

| Plugin | Purpose |
|--------|---------|
| [williamboman/mason.nvim](https://github.com/williamboman/mason.nvim) | Installs & manages LSP servers |
| [williamboman/mason-lspconfig.nvim](https://github.com/williamboman/mason-lspconfig.nvim) | Bridges Mason and nvim-lspconfig |
| [WhoIsSethDaniel/mason-tool-installer.nvim](https://github.com/WhoIsSethDaniel/mason-tool-installer.nvim) | Auto-installs LSP servers and formatters |
| [neovim/nvim-lspconfig](https://github.com/neovim/nvim-lspconfig) | Core LSP configuration |
| [folke/lazydev.nvim](https://github.com/folke/lazydev.nvim) | Neovim Lua API completions |
| [nvim-treesitter/nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) | Syntax highlighting and indentation |
| [saghen/blink.cmp](https://github.com/saghen/blink.cmp) | Autocomplete engine |
| [stevearc/conform.nvim](https://github.com/stevearc/conform.nvim) | Code formatting on save |
| [OXY2DEV/markview.nvim](https://github.com/OXY2DEV/markview.nvim) | Markdown rendering |
| [bullets-vim/bullets.vim](https://github.com/bullets-vim/bullets.vim) | Markdown list continuation and checkbox toggling |
| [antonk52/markdowny.nvim](https://github.com/antonk52/markdowny.nvim) | Markdown bold, italic, code, strikethrough keymaps |

## Instructions for use

```bash
git clone https://github.com/tkdeshpande/neovim-config ~/.config/nvim
```
